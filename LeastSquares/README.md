# Метод наименьших квадратов

Для всех примеров `n = 6`, однако код будет работать корректно для всех `n > 0`
(при условии, что `m >= n`, где `m` - количество _уникальных_ узлов интерполяции `x`).

## Примеры

* in_x = [-3, -2, -1, 0, 1, 2];
* in_y = [10, 2, -3, 7, -4, 1];
